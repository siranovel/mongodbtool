## Usage ##

## install ##

From rubygems:  
~~~
    [sudo] gem install num4distdb
~~~

or from the git repository on github:  
~~~
    git clone https://github.com/siranovel/mongodbtool.git  
    cd mongodbtool  
    jruby -S gem build num4distdb.gemspec
    jruby -S gem install *.gem
~~~
