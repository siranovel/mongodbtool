mongodbtool
===========
commons-math3を利用して、分布表のデータをmongodbに格納ツールです。

## install ##
    git clone https://github.com/siranovel/mongodbtool.git  
    cd mongodbtool/bin  
    任意ディレクトリへコピーし、tarファイルを解凍
